# README

Program Java sederhana untuk menghitung faktorial (N!).\
Program meminta input angka N lalu menghitung hasil faktorial
menggunakan perulangan.

## Cara Menjalankan

1.  Buka project di IntelliJ IDEA.
2.  Buat file Java baru, misalnya `Day65.java`.
3.  Jalankan program dan masukkan nilai N saat diminta.

## Contoh Output

    --- Program Hitung Faktorial (N!) ---
    Masukkan angka N:
    > 5
    Hasil: 120

## Catatan

-   Pastikan input berupa angka positif.
-   Logika menggunakan perulangan dari N ke 1.
